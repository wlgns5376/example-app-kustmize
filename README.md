# kustmize

## Structure
```
├── base
│   ├── configmap.yaml
│   ├── deployment-db.yaml
│   ├── deployment.yaml
│   ├── ingress.yaml
│   ├── kustomization.yaml
│   └── secret.yaml
└── overlays
    └── dev
        ├── configmap.yaml
        ├── ingress.yaml
        ├── kustomization.yaml
        └── secret.yaml
```
## build
```sh
kustomize build overlay/dev/
```

## apply kubernetes
```sh
kustomize build overlay/dev/ | kubectl apply -f -
```
